# “智龄汇” (老年版直聘) 毕业设计演示使用手册

同学你好，欢迎测试“智龄汇”核心功能体验版（Demo）。
为了方便你在毕业答辩/现场展示时，给老师或评委留下最深刻的印象，请按照以下指南进行演示。

---

## 一、项目核心亮点（答辩时重点口述）
本项目是一款转为退休老年人量身打造的“银发族招聘平台”。
在演示时，你可以向答辩老师着重强调以下 **3个技术与设计亮点**：

1. **极致的“适老化”无障碍 UI 设计**
   - 我们**没有**使用复杂的下拉框或多选菜单！全站字号起步 `16px/18px`，采用高对比度的暖橙/青绿双色搭配。
   - 所有交互（比如选择身体状况、技能标签）全部改为**占地面积大、一点即选**的“大号点选卡片”。

2. **真正可演示的“无边界”双端切换体验**
   - 顶部导航栏提供了醒目的【长者版】/【企业版】双端直接切换开关。点击后，一套系统瞬间变色并切换业务流（不仅是界面的改变，而是身份的转换）。

3. **零后端全本地闭环（防依赖/防断网）**
   - 为了答辩环境的绝对稳定性，我们将所有的求职、发布岗位的动作数据，通过前端浏览器本地技术（`LocalStorage`）直接记录。
   - **完全不需要配置复杂的数据库即可向老师展示整套数据的流转**（HR 发出的岗位，老人在页面马上就能刷到）。

## 二、如何打开运行项目 (30秒极简版)

只需在你用来展示的电脑/投影仪上打开浏览器，**直接访问以下网址**即可进行演示：

👉 **[国内主力演示专线]**: https://pipiwolve.github.io/HollywoodCut/

*(注：如果在国内网络如遇打不开、白屏等情况，请开启加速器或更换上述主力链接)*

## 三、答辩现场推荐演示流程（黄金 3 分钟剧本）

你可以按照以下剧本来进行实景演示：

### 🎯 场景一：体验长者找工作 (C端老年人视角)
1. **[点击访问网页]**：默认进入“长者版”岗位大厅（暖橙色温馨主题）。
2. 让大家观察巨大的按钮和清晰的薪资标识。
3. **[点击右上角：我的微简历]**：给老师展示极简的填表体验。
4. **演示亮点：使用“语音录入”！**
   - 点击麦克风按钮，尝试对着电脑说话：“我身体很硬朗，会点维修”。
   - 文本框会自动把你说的话转换成文字填入表中！（展示 Web Speech 现代浏览器 API）
5. 点击左下角所有硬朗与技能的“大号标签”，然后点击保存。

### 🎯 场景二：企业HR视角发布岗位 (B端企业视角)
1. **[在网页最上方中心，点击“企业版”]**。
2. 背景瞬间丝滑切至“商务青绿色主题”，来到此时的“牛人大厅（人才库）”。
3. 可以在这里向老师展示刚刚我们填写的那个长者简历信息。
4. **[点击顶部“发布岗位”]**：快速给表单填写内容：
   - 名称：“社区活动登记员”
   - 薪水：“3500元”
   - 勾选几项福利标签并保存。

### 🎯 场景三：业务闭环与智能聊天模拟 (压轴展示)
1. **[在网页最上方中心，切回“长者版”]**。
2. 告诉老师：“下面我们刚才企业用人方发出的岗位，立马被求职老人看到了”。滚动网页找到刚才加的“社区活动登记员”。
3. **[点击非常醒目的“立即沟通”按钮]**！
4. 页面右下角弹出一个聊天挂件：
   - 你在聊天框里敲上一句：“你好，我有意向。”并发送。
   - **停顿 1 秒重点演示：系统会像真人在对面一样，弹出一个“正在输入”的动画点，随后模拟出真正的对方 HR 回复文本！**

通过以上三步剧本式走查一遍，答辩现场一定会惊呼于这套演示系统的极高完成度与流畅度。
